# CS31007---Assignment-2

## Assignment Brief
In teams of three, your task is to design and develop a Quiz Management System to store and manage questions, answers, scoring rules, and to run quizzes, recording the answers and scoring the results. Your Quiz Management System should include different question types and result calculations.

This focus of this project is designed to assess your understanding and practical application of good software architecture design, design patterns, unit testing and Test-Driven Development (TDD) and defensive programming within a collaborative software development team utilising version control, and shared ownership of code.

You should design and implement a Quiz Management System, that includes the following:
- Develop an architecture design using design patterns where appropriate,
- Apply Test-Driven Development (TDD) within your design,
- Demonstrate a solid, practical understanding of unit testing concepts using a Unit testing framework, with automated test code.
- Design effective unit tests that cover normal, boundary, and exceptional cases,
- Use automated tests to support refactoring and improve code quality
- Critically evaluate the effectiveness of tests and testing strategies
- Work effectively as part of a small development team using professional tools and workflows
- Reflect on your architecture design, development and testing.
